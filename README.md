# downlink-uplink-irs-network
IRS-aided downlink and uplink NOMA/OMA network performance analysis using MATLAB.
##Abstract
This project focuses on Intelligent Reflecting Surface (IRS) aided wireless communication systems for uplink and downlink transmission using Non-Orthogonal Multiple Access (NOMA) and Orthogonal Multiple Access (OMA) techniques. The objective of the project is to improve signal quality, spectral efficiency, network coverage, and communication reliability in next-generation wireless networks. The system performance of NOMA and OMA schemes is analyzed and compared under different communication scenarios using simulation techniques. The project demonstrates how IRS technology can enhance wireless communication performance and optimize network efficiency in modern communication systems.
##Technologies Used:
-MATLAB
-Wireless Communication
-Intelligent Reflecting Surface (IRS)
-NOMA
-OMA
-Signal Processing
-Simulation and Modeling
##Features:
-Analysis of uplink and downlink communication systems
-Implementation of IRS aided wireless networks
-Comparison between NOMA and OMA techniques
-Improved spectral efficiency and signal strength
-Enhanced network coverage and communication reliability
-Performance evaluation using simulation models
##Results:
-Achieved improved signal performance using IRS technology
-NOMA demonstrated better spectral efficiency compared to OMA
-Enhanced wireless communication coverage and throughput
-Reduced signal degradation and improved communication reliability
-Successfully analyzed network performance under different transmission conditions
